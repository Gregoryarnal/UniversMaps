# UniversMaps
Ceci est un projet de carte du système solaire permettant un accès simplifié et "user friendly" aux données sur les differents astres qui nous entourent.

Le projet UniversMaps à pour but d'évoluer vers une carte 3D interactif dans la quelle l'utilisateur pourra librement se déplacer pour admirer les merveilles de l'espace tout en apprennant.

Il en est à sa version 1.0 avec une carte en 2D, le déplacement des planetes est à vitesse réel (1 jours = 1 secondes )
